# Лендинг «Волшебная Керала — страна Богов»

Одностраничный лендинг аюрведа-йога-ретрита в Кералу (10–21 марта 2027).
Статика без сборщиков: `index.html` + `styles.css` + `script.js` — работает на GitHub Pages «из коробки».

Заменяет страницу на telegra.ph: <https://telegra.ph/Volshebnaya-Kerala---strana-Bogov-10-02>

## Структура

```
kerala/
├── index.html      — лендинг
├── styles.css      — стили (CSS-переменные палитры в начале файла)
├── script.js       — счётчик мест, цели Метрики, аккордеоны, анимации
├── oferta.html     — публичная оферта (заглушка, проверить у юриста)
├── privacy.html    — обработка персональных данных (заглушка)
├── favicon.svg
└── img/            — фото в WebP (скачаны и сжаты из исходника)
```

## Как поменять количество мест

В начале `script.js`: переменная `PLACES_LEFT`.
